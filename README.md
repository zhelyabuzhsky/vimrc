# My vimrc

Inspired by [The Ultimate vimrc](https://github.com/amix/vimrc)

## How to install

    $ ln -rs .vimrc ~/.vimrc

## Key Mappings

Open new tab:

    , + t + n

Close tab:

    , + t + c

Next tab:

    , + t + n

Open a new tab with the current buffer's path:

    , + t + e

Switch CWD to the directory of the open buffer:

    , + c + d

Toggle paste mode on and off:

    , + p + p
